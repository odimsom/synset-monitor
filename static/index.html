Diagnósticos Inteligentes
  • ¿Qué es IO Wait? Indica el tiempo que el procesador está inactivo esperando a que un disco conteste. Si ves una carga de sistema ralentizada pero el CPU usage normal, el problema es IO Wait Alto.
  • ¿Qué es Swap Paging? Cuando la RAM física está llena, el sistema operativo usa el disco (Swap). Un volumen alto de Swap IN / OUT constantes es una emergencia de rendimiento.
  • ¿Qué es CPU Steal Time? Exclusivo de servidores virtuales (VPS / Cloud). Es la cantidad de tiempo de CPU hipervisor que los hostings te están robando o dando a otras máquinas vecinas.
